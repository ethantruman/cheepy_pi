# cheepy_pi

A budget-friendly, fully opensource DIY mini Cyberdeck powered by Raspberry Pi Model B, featuring a custom BQ25894+TPS61236P UPS, 3 cheap ST7789 display and audio system.

## Features
* **Processor:** Powered by Raspberry Pi Model B.
* **Power System:** Custom UPS utilizing BQ25894 + TPS61236P.
* **Display:** Triple (3x) low-cost ST7789 screens.
* **Audio:** Integrated audio system.
